## Requirements
* node `^4.2.0`
* npm `^3.0.0`


## Development

```bash
$ git clone https://github.com/Hikaru-Ito/react-redux-music-app
$ cd ./react-redux-music-app
```

install modules

```bash
$ npm install
```

run gulp

```bash
$ gulp
```

## Production

after clone & install modules

```bash
$ npm run build
$ npm start
```
